# 主要功能

- 提供查询服务，具体可查询的内容取决于我们的数据库，只要我们的数据库足够丰富，可查询的内容就很多
- 添加评论功能
- 代码高亮
- 可根据用户使用评星在主页面
- 页面整洁，ui统一
- 可拓展性前，代码可重用性高
